Report: Simulating High-Frequency American Options Data

1. Objective

The purpose of this simulation was to generate high-frequency tick-level options data (with updates every second) for research purposes. The dataset reflects realistic market dynamics based on historical benchmarks and includes information such as option prices, underlying prices, Greeks, implied volatility, and bid-ask spreads.

2. Methodology

2.1 Underlying Asset Simulation

The underlying asset’s price was simulated using a Geometric Brownian Motion (GBM) model:

[
S_{t+\Delta t} = S_t \cdot \exp\left((\mu - 0.5\sigma^2)\Delta t + \sigma\sqrt{\Delta t}Z\right)
]

Where:

  • (S_t): Current underlying price.
  • (\mu): Annualized drift (expected return).
  • (\sigma): Annualized volatility.
  • (\Delta t): Time step size (1 second in this case).
  • (Z): Standard normal random variable.

Parameters:

  • Starting Price: $100
  • Drift: 5% annualized.
  • Volatility: 20% annualized.
  • Time Step: One second.

This process ensures realistic price evolution over a one-hour trading window (3600 seconds).

2.2 Option Price Calculation

Option prices were calculated using the Black-Scholes Formula, which provides closed-form solutions for European-style options. While American options allow early exercise, this simulation approximates their prices using the Black-Scholes method.

The formula for a call option is:
[
C = S \cdot N(d_1) - K \cdot e^{-rT} \cdot N(d_2)
]
And for a put option:
[
P = K \cdot e^{-rT} \cdot N(-d_2) - S \cdot N(-d_1)
]

Where:

  • (d_1 = \frac{\ln(S / K) + (r + 0.5\sigma^2)T}{\sigma\sqrt{T}})
  • (d_2 = d_1 - \sigma\sqrt{T})
  • (N(\cdot)): Cumulative distribution function of the standard normal distribution.
  • (S): Current underlying price.
  • (K): Strike price.
  • (T): Time to expiration (in years).
  • (r): Risk-free rate (assumed to be 5%).
  • (\sigma): Implied volatility (assumed constant at 20%).
2.3 Strike Prices and Expirations
  • Strike prices: $95, $100, and $105.
  • Expiration: December 20, 2024.
  • Time to expiration ((T)) was dynamically calculated for each tick.
2.4 Bid-Ask Spread Simulation

To simulate market microstructure:

  • A random bid-ask spread was added to the calculated option prices.
  • Spread values were uniformly sampled between $0.01 and $0.05.
  • Bid price = (\text{Option Price} - \text{Spread})
  • Ask price = (\text{Option Price} + \text{Spread})
2.5 Data Fields

For each second, the dataset includes:

  • Timestamp: Precise second of the simulated trading day.
  • Underlying Price: Simulated using GBM.
  • Option Type: Call or Put.
  • Strike Price: Predefined values ($95, $100, $105).
  • Expiration Date: December 20, 2024.
  • Option Price: Calculated using the Black-Scholes model.
  • Bid/Ask Prices: Derived from simulated bid-ask spreads.
  • Implied Volatility: Fixed at 20%.

3. Assumptions and Limitations

  1. European Approximation: While American options allow early exercise, their prices were approximated using the Black-Scholes model for simplicity.
  2. Static Implied Volatility: The implied volatility was fixed at 20%. Real-world volatility often fluctuates intraday.
  3. Risk-Free Rate: Assumed constant at 5%.
  4. No Transaction Costs or Liquidity Constraints: These were excluded for simplicity.
  5. Uniform Bid-Ask Spreads: Real-world spreads depend on factors like option moneyness and market liquidity.

4. Conclusion

The simulation provides a realistic representation of high-frequency options data for a single trading hour. The dataset includes second-by-second updates with dynamically calculated option prices, Greeks, and bid-ask spreads. While some assumptions simplify real-world complexities, the data can serve as a robust foundation for research or algorithmic trading experiments.

If further refinements or customizations (e.g., dynamic implied volatility or jump diffusion modeling) are needed, additional steps can be implemented.
